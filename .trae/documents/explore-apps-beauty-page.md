# 探索页改为 AI+化工/化妆品 主题装饰页面

## 动机

当前 `/explore/apps` 页面为空白的 `<div />`，不美观。用户希望展示一个 **AI + 化工/化妆品** 主题的装饰性页面，保留探索页面的侧边栏功能，但内容区域只做品牌展示，没有功能交互。

## 设计方案

### 页面布局

```
┌──────────────────────────────────────────────────┐
│  左侧栏 (Sidebar)          │  内容区域              │
│  ┌────────────────────┐    │  ┌────────────────┐   │
│  │ 已安装的应用列表...  │    │  │  背景装饰区     │   │
│  │                    │    │  │  - 渐变背景     │   │
│  │                    │    │  │  - 分子结构图    │   │
│  │                    │    │  │  - 六边形网格    │   │
│  │                    │    │  │  - AI 粒子效果   │   │
│  │                    │    │  │  - 品牌标语      │   │
│  └────────────────────┘    │  └────────────────┘   │
└──────────────────────────────────────────────────┘
```

### 视觉元素

1. **渐变背景**：浅蓝→淡紫渐变（呼应化工科技感 + 化妆品美学）
2. **装饰性六边形网格**：化学分子结构风格的六边形图案（SVG/CSS 实现）
3. **AI 粒子/圆点装饰**：漂浮的小圆点营造科技感
4. **品牌标语**：居中显示简洁的主题文字，如"AI 赋能化工 · 智创未来"或类似
5. **分子式风格的 SVG 图标装饰**：点缀在页面四周

### 配色方案

- 主色：蓝色系 (#4F46E5 → #0EA5E9) — 化工科技感
- 辅助色：淡紫 (#C4B5FD) — 化妆品柔美感
- 文字：渐变或柔和色调

   - 居中品牌标语文字
