# Dify Plugin Development Guide

This plugin follows the same structure as the other Ace Data Cloud tool provider plugins
in this repository (e.g. nano-banana, veo).

## Run

1. Copy `.env.example` to `.env` and configure your Dify debug environment.
2. Install deps:
   ```bash
   pip install -r requirements.txt
   ```
3. Start:
   ```bash
   python -m main
   ```
