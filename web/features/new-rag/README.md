# New RAG

Dataset 2.0 knowledge-space listing and empty knowledge creation backed by the generated Console API contract.

## Internal Modules

- `create-knowledge-space-dialog`: Owns the empty knowledge creation form and cache refresh behavior.

## External Modules

- `app/components/base/skeleton`: Provides scoped loading placeholders for the knowledge-space grid.
- `hooks/use-document-title`: Keeps the browser title aligned with the Knowledge surface.
