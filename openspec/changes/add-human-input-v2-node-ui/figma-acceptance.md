## Shared visual constraints

- Use existing workflow typography, semantic color tokens, `rounded-lg` for Figma's 8px radius and dify-ui overlay/control primitives.
- The configuration surface fills the workflow panel. Measured `388px` controls become `width: 100%` below that inner width and must not create horizontal scrolling.
- Error text is semantic and announced; focus rings remain visible. Tooltip-only information also has an accessible name.
- Node recipient summaries distinguish empty, initiator-only, contacts-only, mixed, overflow and invalid/unresolved data. The four referenced cards are the primary snapshots; overflow and invalid are deterministic harness states.
- All strings are localized. Components contain no English product copy.
