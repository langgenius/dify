## Responsive and Accessibility

- Account Settings 继续使用现有 breakpoint 行为：桌面显示 224px setting menu，窄屏收敛为 44px icon rail，内容区域保持可滚动。
- 520px binding dialog 与 840px sync-details dialog 使用 viewport-aware max width / max height；小屏内部滚动，不增加 call-site z-index。
- 所有 overlay 使用 `@langgenius/dify-ui/dialog` 或 destructive confirmation 使用 `@langgenius/dify-ui/alert-dialog`。
- button、filter、form field 和 close control 必须有 visible focus、accessible name、键盘提交和正确错误关联。
